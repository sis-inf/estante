# Estante

Gestor de base de datos con interfaz de línea de comandos en Java que permite ejecutar consultas SQL sobre PostgreSQL.

## ¿Qué es?
Estante es una aplicación de consola desarrollada en Java que permite al usuario interactuar con una base de datos PostgreSQL mediante consultas SQL.

## ¿Para quién es?
Está dirigido a estudiantes y desarrolladores que desean aprender o practicar el uso de bases de datos utilizando Java y JDBC.

## ¿Qué problema resuelve?
Facilita la ejecución de consultas SQL desde una aplicación Java sin necesidad de utilizar herramientas externas, integrando la lógica de negocio con el acceso a datos.

## Instalación

### Requisitos
- Java JDK 17
- Maven
- PostgreSQL

### Pasos

1. Clonar el repositorio
2. Compilar el proyecto con Maven:
   mvn clean install
3. Ejecutar la aplicación:
   mvn exec:java

## Tecnologías
- Java
- PostgreSQL
- JDBC
- Maven
