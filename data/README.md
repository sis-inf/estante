# Directorio `data`

Este directorio organiza archivos relacionados con actividades de análisis y procesamiento de datos dentro del proyecto.

## Estructura

### `raw/`

Contiene datos originales sin modificaciones que pueden servir como punto de partida para tareas de análisis o transformación.

### `processed/`

Contiene datos que han sido preparados o transformados para su uso en procesos posteriores de análisis o procesamiento.

### `analysis/`

Contiene reportes, gráficos, dashboards u otros resultados generados a partir del análisis de datos.

## Nota

Los datos de ejemplo utilizados para demostrar o probar el funcionamiento de la aplicación se encuentran en el directorio `examples/`. El directorio `data/` está destinado a organizar archivos relacionados con el análisis y procesamiento de datos dentro del proyecto.
