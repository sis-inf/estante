# Telemetría de uso

## ¿Qué es la telemetría?

La telemetría de uso es una funcionalidad opcional que permite recopilar información sobre el uso de las funcionalidades de la aplicación. Su objetivo es ayudar a identificar qué características son utilizadas con mayor frecuencia para mejorar el proyecto.

## Información que se recopila

Cuando la telemetría está habilitada, únicamente se recopilan datos relacionados con el uso de las funcionalidades de la aplicación, por ejemplo:

- Uso de determinadas funcionalidades.
- Frecuencia de utilización de características.
- Eventos relacionados con el funcionamiento de la aplicación.

## Información que NO se recopila

La telemetría **no** recopila:

- Datos de conexión.
- Consultas realizadas por el usuario.
- Resultados obtenidos por el usuario.
- Información personal o sensible.

## Telemetría opcional (Opt-in)

La telemetría es completamente **opt-in**, lo que significa que está deshabilitada por defecto y solo se activa cuando el usuario decide habilitarla de forma explícita.

## Objetivo

La información recopilada se utiliza únicamente para comprender el uso de las funcionalidades y orientar futuras mejoras del proyecto, respetando la privacidad de los usuarios.
