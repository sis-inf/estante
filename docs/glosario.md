# Glosario del Proyecto Estante

## Términos

| Término | Definición |
|---|---|
| Base de datos | Conjunto organizado de información almacenada para su consulta y gestión. |
| Clave foránea | Campo que crea una relación entre dos tablas. |
| Clave primaria | Identificador único de cada registro en una tabla. |
| Conexión | Enlace entre la aplicación Java y PostgreSQL mediante JDBC. |
| Consulta SQL | Instrucción utilizada para interactuar con la base de datos. |
| DDL | Lenguaje para definir estructuras como tablas y esquemas. |
| DML | Lenguaje para manipular datos (insertar, actualizar, eliminar). |
| Driver | Componente que permite la comunicación entre Java y la base de datos. |
| Driver JDBC | Controlador específico que permite la conexión entre una aplicación Java y un sistema gestor de bases de datos mediante JDBC. |
| Esquema | Organización lógica de tablas dentro de la base de datos. |
| Exporter | Módulo encargado de exportar datos o resultados a formatos como CSV o JSON. |
| Favorito | Consulta o script SQL guardado por el usuario para reutilizarlo rápidamente. |
| Generación de DDL | Proceso de creación automática de sentencias SQL para definir estructuras de base de datos. |
| Historial de Queries | Registro de las consultas ejecutadas previamente por el usuario dentro de la aplicación. |
| Importación CSV | Proceso de carga de datos desde archivos CSV hacia una base de datos. |
| JDBC | API de Java que permite conectarse a bases de datos. |
| Motor de base de datos | Software encargado de almacenar, gestionar y recuperar la información de una base de datos. |
| ORM | Técnica que mapea objetos de Java a tablas de base de datos. |
| Panel de estadísticas  | Sección de la aplicación que muestra métricas e información resumida sobre la base de datos.|
| PreparedStatement | Objeto de JDBC que permite ejecutar consultas con parámetros. |
| Registro | Fila dentro de una tabla de base de datos. |
| Resaltado de sintaxis  | Funcionalidad que aplica colores y formatos al código SQL para facilitar su lectura. |
| ResultSet | Objeto que contiene los resultados de una consulta SQL. |
| Tabla | Estructura que almacena datos en filas y columnas. |
| Validación de SQL | Proceso de verificación de sentencias SQL para detectar errores antes de su ejecución. |
